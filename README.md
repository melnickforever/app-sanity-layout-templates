# Portfolio Layout Templates

Generated by AI. Static portfolio website templates built with **Next.js 16**, **React 19**, **Tailwind CSS 4**, and **TypeScript**. Two design variants — minimal and feature-rich — each exported as fully static HTML.

## Layouts

### v1 — Minimal

Clean, whitespace-driven design with monochrome palette.

- **Home** — Hero with `</>` icon + 3 navigation cards (About, Portfolio, Contact)
- **Portfolio** — 2-column project grid with clickable cards
- **Portfolio Detail** — Dynamic `[slug]` pages with overview, challenges, results, and sidebar
- **About** — Bio, skills grid, experience timeline, education
- **Contact** — Form + contact info sidebar with social links

### v2 — Feature-Rich

Gradient hero sections, stats counters, service cards, and more visual elements.

- **Home** — Gradient hero with stats, services section, featured projects, CTA
- **Portfolio** — 2-column project grid with clickable cards
- **Portfolio Detail** — Dynamic `[slug]` pages with overview, challenges, results, and sidebar
- **About** — Photo + bio, skills, experience, education
- **Contact** — Form + contact info sidebar with social links

## Getting Started

### Prerequisites

- Node.js 20+
- npm

### Install & Run

```bash
# Choose a layout version
cd layouts/v1   # or layouts/v2

# Install dependencies
npm install

# Development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

### Build Static HTML

```bash
npm run build
```

Static files are generated in the `out/` directory.

### Routes

| Path | Page |
|---|---|
| `/` | Home |
| `/about` | About |
| `/portfolio` | Portfolio list |
| `/portfolio/[slug]` | Portfolio detail (6 projects) |
| `/contact` | Contact |

Portfolio detail pages are statically generated at build time via `generateStaticParams`.

## Design Tokens

Both layouts use CSS custom properties defined in `globals.css`:

```css
:root {
  --background: #ffffff;
  --foreground: #1a1a1a;
  --primary: #1a1a1a;
  --muted: #777777;
  --card-border: #e5e5e5;
  --section-alt: #fafafa;
}
```

These are mapped to Tailwind via `@theme inline` for use as utility classes like `text-foreground`, `bg-section-alt`, `border-card-border`, etc.
